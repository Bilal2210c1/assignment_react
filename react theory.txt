QUESTION NO 3.
ANSWER :
Simple Component : simple component kisi properties k bagair sirf static value render karne mein use hota hai yah sirf UI display karnay mein use hota

yeh dono aik dusry se isliye alag hain simple mein ap sirf simple ui render karne k liye use kartay ho , jab k parametric mein ap dynamic content render kartay ho as depend on output for UI
simple component 1 hi output de sakta jab k parametric component different input k sath sath different output bhi de sakta hai.
